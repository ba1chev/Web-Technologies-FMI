Тема на проекта:
  Компютърни кешове — задълбочен преглед (йерархия на паметта, политики,
  хронология, сравнителни данни).

Кратко описание:
  Многостраничен статичен сайт само с HTML5, без CSS и JavaScript. Съдържанието
  обяснява ролята на кешовете между CPU и RAM, основни термини (линия на кеш,
  асоциативност, write-back/write-through, MESI), исторически етапи, таблица
  с ориентировъчни параметри по нива и демонстрационна контактна форма.

Файлова структура:
  index.html, about.html, timeline.html, data.html, contact.html
  images/     — векторна схема на йерархията
  media/      — демонстрационен WAV файл за елемента audio

Източници на изображения и медия:
  images/hierarhiya-keshove.svg — оригинална векторна схема, създадена за
    този проект (без външни stock изображения).
  media/cache-explainer.wav — синтетичен аудио сигнал, генериран локално
    за демонстрация на <audio controls>.

Допълнителни референции за текстовото съдържание (за самостоятелно четене):
  Hennessy, Patterson — Computer Architecture: A Quantitative Approach
  Документация на производители на процесори (микроархитектурни обзори)
  MDN Web Docs — HTML елементи и достъпност
